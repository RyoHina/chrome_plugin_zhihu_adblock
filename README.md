# 开发模式使用：
1. 将Releases中"Source code.zip"，随便解压缩到一个目录。
2. 打开Chrome设置->扩展程序，点击右上角开启"开发者模式"。
3. 点击左侧"加载已解压的扩展程序"，选择第一步中的解压缩目录即可。


打开知乎首页，F12打开Console
